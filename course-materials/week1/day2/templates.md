## Чек-лист для самопроверки: Готовность к AI-аудиту

Используйте этот чек-лист, чтобы убедиться, что вы учли все ключевые аспекты при анализе процессов.

### Фаза 1: Подготовка и картирование

- [ ] Выбран конкретный, измеримый бизнес-процесс для анализа.
- [ ] Определен владелец процесса и ключевые участники.
- [ ] Процесс описан текстом «как есть» (as-is).
- [ ] Создана BPMN-схема процесса, включающая основные развилки (шлюзы).
- [ ] Проведена декомпозиция ключевых задач на элементарные действия.

### Фаза 2: Оценка и приоритизация

- [ ] Процесс оценен по всем критериям влияния (Время, Деньги, Качество, Масштаб).
- [ ] Процесс оценен по всем критериям сложности (Техническая, Организационная, Данные).
- [ ] Оценки аргументированы и основаны на фактах или экспертных предположениях.
- [ ] Рассчитаны итоговые баллы и определено положение процесса на матрице приоритизации.
- [ ] Сделан вывод о стратегическом подходе к автоматизации данного процесса (делать, планировать, делать по возможности, избегать).

### Фаза 3: Формирование бэклога

- [ ] Сформулирована четкая User Story, описывающая цель автоматизации.
- [ ] Определены конкретные, измеримые критерии приемки.
- [ ] Карточка задачи готова для обсуждения с командой разработки или руководством.
